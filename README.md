# onlineclock

An interactive HTML app that helps students learn to tell time using both **analog** and **digital** clocks.

## Features

- **Live Clock** – real-time analog clock (hour, minute, and second hands) with a matching digital display in 12-hour format.
- **Practice Mode** – the clock displays a random time (rounded to 5-minute intervals). Students type the time they read and get instant feedback, including the correct answer if they are wrong.
- No dependencies – a single `index.html` file, works offline in any modern browser.

## Usage

Open `index.html` in a web browser.

- Use the **Live Clock** button to watch the current time.
- Use the **Practice Mode** button to test your time-reading skills.
  - Enter the time shown on the clock (e.g. `3:45` or `3:45 PM`) and press **Check ✓**.
  - Press **New Time ↺** to generate a different time to practise with.
